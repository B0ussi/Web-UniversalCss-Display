Description

This website is comprised of 3 different fonts. The first of the bunch is the Titan One font. This font is the beefiest, and thus is used in the main h1 tags. The second of these fonts is the Chewy font. This font is just thrown in the other headers. To be honest, I don't think that it looks all that good, but yes it's still there. The last of these fonts is the font you're reading right now. This is the DynaPuff font. Yes, only one word. I think this font is the most pleasing to read, so it is the font that is used in paragraphs.
